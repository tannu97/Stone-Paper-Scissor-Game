# Stone-Paper-Scissor Game

A simple interactive game built with HTML, CSS, and JavaScript. Users can play against the computer with real-time score tracking and responsive design.

## Tech Stack
- HTML, CSS, JavaScript

## How to Play
1. Open `index.html` in your browser.
2. Select Stone, Paper, or Scissor.
3. Play against the computer and track your score.
